### 2022 – *iPad Deployments Using Mosyle Manager and Custom Tools*
An overview of the district's migration from JAMF Pro to Mosyle Manager in 2020. The presentation detailed their use of Mosyle's "Limbo mode" for pre-configuration, managing device breakage and loaners, and introduced MOSBasic, a custom command-line tool he developed to automate tasks via the Mosyle API. It also covered using CFGUtil to efficiently re-image and reprovision iPads.

This presetation was done during the Campfire Session Years.  My first Virtual/Remote presentation done with my co-presenter Jennifer Czyzewski

### 2019 – *iPad Evolution at Gateway School District*
 A case study of Gateway School District's iPad deployments, detailing the mistakes from their initial 1:1 rollout in 2014 and the lessons applied to their successful 2017 redeployment. The talk covered their evolution to K-8 1:1 and large-scale Shared iPad deployments in elementary schools, including topics like Apple TV deployment and using the JAMF Pro API

  Slides -> https://macadmins.psu.edu/files/2019/07/psumac2019-389-iPad-Evolution-at-Gateway-School-District.pdf
